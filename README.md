# Twin Beans Coffee

Website for Twin Beans Coffee, a Northwest Indiana café combining specialty coffee with a relaxed, community-focused atmosphere.

Live: https://twinbeansnwi.com

---

## Overview

Designed and developed a marketing website to reflect the café’s unique brand, highlight offerings, and provide essential visitor information.

## Features

- Responsive design for mobile and desktop  
- Menu and location information  
- Brand-focused layout and visuals  

## Tech Stack

- Laravel (PHP)  
- Bootstrap 5  
- JavaScript  

## Role

Full design and development, including layout, frontend implementation, and deployment.
